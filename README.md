# Come avviare il progetto

## Metodo più semplice

1. Scarica tutti i file
2. Apri `index.html`
3. La mappa si aprirà direttamente nel browser

## Metodo consigliato (VS Code)

1. Apri la cartella in VS Code
2. Installa l'estensione "Live Server"
3. Tasto destro su index.html
4. "Open with Live Server"

## Personalizzazione

Nel file `script.js` puoi:
- aggiungere nuovi luoghi
- cambiare coordinate
- sostituire immagini
- modificare descrizioni

## Coordinate

Puoi trovare coordinate facilmente su Google Maps:
clic destro → "Che cosa c'è qui?"
